# 📱 Curso-Flutter

Repositório utilizado para acompanhar o curso **[Curso Completo de Criação de Apps com Flutter (Android, iOS, Web e Desktop)](https://www.udemy.com/course/curso-completo-flutter-app-android-ios/)** ministrado por **Daniel Ciolfi**, **Ewerton Dutra**, **Startto.dev | Cursos Profissionais**.

---

## 🎯 Objetivo do Curso
O curso ensina **Flutter do básico ao avançado**, cobrindo desde a linguagem Dart até a criação de aplicativos completos para **Android, iOS, Web e Desktop**.  
O foco é preparar o aluno para se tornar um desenvolvedor completo, capaz de criar apps multiplataforma com apenas um código, utilizando os recursos mais modernos do Flutter e Dart (com Null-Safety).

---

## 📚 Conteúdos Abordados
Durante o curso, foram desenvolvidos e aplicados os seguintes conhecimentos:

- Fundamentos da linguagem **Dart** (básico, intermediário e orientação a objetos);
- Null-Safety no Dart;
- Estrutura e configuração de projetos Flutter;
- Criação de interfaces modernas e responsivas com **Widgets**;
- Programação assíncrona e manipulação de dados;
- Armazenamento local e persistência de informações;
- Integração com APIs e comunicação cliente-servidor;
- Navegação entre telas e gerenciamento de estado;
- Publicação e compartilhamento de aplicativos;
- Exercícios práticos e projetos completos para portfólio.

---

## 🛠️ Tecnologias Utilizadas

4. **Flutter002 (2022)** – Aplicativo Completo de Lista de Tarefas
